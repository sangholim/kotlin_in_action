### object 키워드: 클래스 선언과 인스턴스 생성

- 4.4.1
    - 객체 선언 - 싱글턴 정의하는 방법중 하나
    - 동반 객체 - 어떤 클래스와 관련된 팩토리 메서드 관리
    - 객체 식 - 자바의 무명 내부 클래스 대신 사용
    - `object`
        - 생성자 선언부가 없다.
        - 메서드, 프로퍼티 접근시 예제
          `A.method()`
        - 클래스, 인터페이스 상속 가능
        - 클래스 내부에 `object` 생성 가능
    - [lesson-4.4.1](lesson4.4.1.kt)
